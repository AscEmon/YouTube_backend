# Youtube Backend

## ERD: https://app.eraser.io/workspace/6ajCLoTFUHIbVXZ0ipYq?origin=share

### Built with Node.js, Express.js, MongoDB, Mongoose, JWT, Bcrypt, and numerous other technologies, this project is a sophisticated backend project. With every feature a backend project ought to have, this one is comprehensive. We are creating a comprehensive video hosting website that is comparable to YouTube and has all of its features, including the ability to log in, register, upload videos, like, dislike, comment, reply, subscribe, unsubscribe, and many more.
